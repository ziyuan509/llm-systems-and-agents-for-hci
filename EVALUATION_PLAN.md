# Evaluation Plan

## Goal

Test whether a governance-oriented interface helps ordinary users manage a deep research agent better than simpler control conditions.

## Candidate Conditions

- Condition A: black-box chat agent
- Condition B: trace-oriented oversight view
- Condition C: delegation-contract interface

## Candidate Tasks

- source gathering under trust constraints
- claim verification with conflicting evidence
- report generation under budget and time limits

## Candidate Measures

- intervention success
- final answer quality
- time to correction
- calibrated reliance
- perceived control
- mental workload

## Logging Priorities

- when users revise policy
- what triggers ask-back moments
- when users override the agent
- whether repair happens at the step level or policy level
