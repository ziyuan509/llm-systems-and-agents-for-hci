## Featured Project

`Delegation Contracts for Privacy-Sensitive Pervasive AI Agents`

## One-line Summary

A governance-first interface prototype that helps ordinary users define, revise, and repair the boundaries of AI agents that sense, remember, act, and disclose information in shared physical spaces.

## Why This Project Matters

- It moves beyond black-box chat and raw trace debugging.
- It focuses on user-governable delegation boundaries for ambient AI.
- It turns privacy, memory, disclosure, action, and escalation rules into user-facing interaction objects.
- It connects pervasive AI, bystander privacy, and end-user control into a concrete HCI prototype.
- It frames agent governance as an interaction design problem rather than only a backend safety or policy problem.

## Current Artifact

The current prototype demonstrates a shared-space ambient AI scenario where a user configures and repairs a delegation contract for a meeting assistant.

The prototype covers:

- task setup
- delegation contract drafting
- runtime ask-back
- bystander privacy risk
- contract revision
- safe rerun
- governance audit summary
- session event logging for pilot walkthroughs
