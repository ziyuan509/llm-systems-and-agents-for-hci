# Roadmap

## Phase 1: Scope the Research Project

- lock the project to `Delegation Contracts for Privacy-Sensitive Pervasive AI Agents`
- define the main claim and non-goals
- finalize the shared-space ambient AI task domain and baseline conditions

Status: mostly complete.

## Phase 2: Specify the System

- define the delegation contract model
- define the UI states and user flow
- define escalation triggers and repair actions

Status: mostly complete for the first prototype.

## Phase 3: Build the MVP

- implement a minimal ambient AI governance loop
- expose editable policy controls for sensing, memory, disclosure, actions, confidence, and escalation
- support at least one runtime ask-back moment
- support at least one policy repair flow

Status: first static demo implemented in `agents/delegation-contract-ui/demo/`.

## Phase 4: Prepare Research Evidence

- refine related-work positioning
- run 3-5 pilot walkthroughs
- prepare setup, ask-back, repair, and audit screenshots
- organize exported logs and qualitative notes into early findings

Status: in progress.

## Phase 5: Anonymous Workshop Package

- draft a 4-page anonymized short paper
- remove author-identifying information from screenshots and materials
- document prototype decisions, limitations, and pilot protocol

Status: next focus.
