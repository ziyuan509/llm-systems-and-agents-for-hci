# Decisions

- Primary framing: end-user agent governance
- Core interaction object: delegation contract
- First domain: privacy-sensitive ambient AI in shared spaces
- Primary user group: ordinary users, not agent developers
- Main contrast: black-box ambient assistants and oversight-only privacy markers
